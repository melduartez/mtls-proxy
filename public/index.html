🔐 mTLS Proxy

Um proxy seguro para requisições com certificados mTLS (mutual TLS). Perfeito para comunicação segura entre serviços.

📍 Endpoint Principal

POST /api/mtls

📤 Formato da Requisição

{
  "url": "https://api.example.com/endpoint",
  "method": "GET",
  "headers": {
    "Authorization": "Bearer token"
  },
  "body": "",
  "cert": "-----BEGIN CERTIFICATE-----...",
  "key": "-----BEGIN PRIVATE KEY-----..."
}

📥 Formato da Resposta

{
  "status": 200,
  "statusText": "OK",
  "headers": {...},
  "body": "response body"
}

🔑 Autenticação

Todas as requisições requerem um header Authorization: Bearer <MTLS_PROXY_SECRET>
